# دعوة تخرّج كايلا — Kayla Graduation Invitation

دعوة تخرّج تفاعلية بصفحة واحدة، عربية بالكامل، مبنية على لوحة مائية بنسبة 9:16 مُقسَّمة إلى
طبقات مستقلة تتحرك كل واحدة منها على حدة.

**الرابط:** https://samarnatsheh.github.io/kayla-graduation/

---

## ✏️ ما الذي تعدّلينه

| ماذا | أين |
|---|---|
| **كل النصوص العربية** (الاسم، الأبيات، التاريخ، الوقت، المكان، الختام) | `lib/invitationContent.ts` |
| **رابط الموقع على الخرائط** | `lib/config.ts` ← `MAP_URL` |
| **ملف الموسيقى** | `lib/config.ts` ← `AUDIO_SRC` |

### رابط الخرائط

في `lib/config.ts`:

```ts
export const MAP_URL = 'REPLACE_WITH_FINAL_GOOGLE_MAPS_LINK';
```

ما دام على هذه القيمة، تبقى بطاقة **المكان** ظاهرة بالنص فقط بدون زر — لا رابط مكسور.
بمجرد لصق رابط حقيقي تتحوّل البطاقة إلى زر يفتح الخريطة في تبويب جديد.

### الموسيقى

لا يوجد ملف صوتي في هذه النسخة. بعد وضع ملف mp3 مرخّص في
`public/audio/kayla-invitation.mp3`، غيّري في `lib/config.ts`:

```ts
export const AUDIO_SRC = '/audio/kayla-invitation.mp3';
```

عندها فقط يظهر زر الصوت. وما دامت القيمة `null` لا يُعرض الزر إطلاقًا — بلا مشغّل مكسور
وبلا طلبات فاشلة في الـ console.

## 🎨 كيف بُنيت الطبقات

اللوحة الأصلية `941×1672` قُصّت إلى **20 طبقة** موضوعة كل واحدة في مكانها الأصلي بالضبط،
فوق ورق عاجي نظيف خالٍ من أي زخرفة — فلا يوجد نسخة ثابتة تحت الطبقات المتحركة.

- `scripts/layers.mjs` — إحداثيات كل طبقة ومقدار تنعيم حوافها
- `scripts/extract-layers.mjs` — القص والتصدير (`npm run assets`)
- `scripts/audit-coverage.mjs` — تدقيق التغطية (`npm run audit:coverage`)

حواف كل طبقة تذوب عبر أقنعة CSS، فلا يظهر أي حدّ مستطيل. آخر تدقيق تغطية: **0.16٪**
من اللوحة غير مغطّى، ولا توجد أي كتلة متبقّية أكبر من 30 بكسل.

الخطوط محمّلة محليًا (`scripts/fetch-fonts.mjs` → `app/fonts.css`) حتى لا يعتمد البناء
على الوصول إلى Google Fonts.

> **ملاحظة:** اللوحة الأصلية `scripts/source-artwork.png` غير مرفوعة إلى المستودع العام.
> أعيديها إلى مكانها إذا أردتِ إعادة توليد الطبقات.

## 🛠 التشغيل والبناء

```bash
npm install
npm run dev                      # تطوير على http://localhost:3000
NEXT_PUBLIC_BASE_PATH=/kayla-graduation npm run build   # تصدير ثابت إلى out/
```

النشر: محتويات `out/` تُرفع إلى فرع `gh-pages` مع ملف `.nojekyll`.

## ♿ إمكانية الوصول

`lang="ar"` و `dir="rtl"`، عناوين دلالية، تسميات عربية لكل زر، ودعم كامل لـ
`prefers-reduced-motion` (يُعرض المحتوى كاملًا بلا أي حركة).
